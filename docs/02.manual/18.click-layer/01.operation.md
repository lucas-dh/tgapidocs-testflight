#### 参数描述

| 属性                  | 类型    | 说明                                                                                                   | 端渲染 | 流渲染 |
| --------------------- | ------- | ------------------------------------------------------------------------------------------------------ | ------ | ------ |
| id                    | String  | 图层对象 id，新建时调用者自己传入的唯一标识，用于各种操作的对象识别                                    | 支持   | 支持   |
| name                  | String  | 图层名称，支持为图层自定义名称                                                                         | 支持   | 支持   |
| coordType             | Number  | XY 轴坐标类别，0：X 经度，Y 纬度；1：X 米，Y 米，相对世界 00 点                                        | 支持   | 支持   |
| coordTypeZ            | Number  | Z 轴坐标类别，0：相对 3D 世界 0 海拔；1：相对 3D 世界地表；2：相对 3D 模型表面（单位：米）             | 支持   | 支持   |
| visible               | Boolean | 添加当前图层时默认显示还是隐藏                                                                         | 支持   | 支持   |
| tags                  | Array   | 用户自定标签数组，用于保存用户的扩展数据，或者可以控制显隐                                             | 不支持 | 支持   |
| legends               | Array   | 定义图层包含图例，支持为不同图例定义各自样式                                                           | 支持   | 支持   |
| legends[0].name       | String  | 图层内图例名称，新建时调用者自己传入的唯一名称，用于各种操作的图例识别                                 | 支持   | 支持   |
| legends[0].icon       | String  | 内置图标名称，或根据项目，在场景中预置好的图标对象的名称，包含文字样式定义，具体图标名称参考下表和下图 | 支持   | 支持   |
| legends[0].iconScale  | Number  | 默认值是 1 表示原大，0 - 1：表示缩小倍率，大于 1：表示放大倍率                                         | 支持   | 支持   |
| legends[0].labelScale | Number  | 默认值是 1 表示原大，0 - 1：表示缩小倍率，大于 1：表示放大倍率                                         | 支持   | 支持   |
| legends[0].lineColor  | String  | 路线颜色（HEX 颜色值）                                                                                 | 支持   | 支持   |
| legends[0].lineWidth  | Number  | 路线宽度（单位：米）                                                                                   | 支持   | 支持   |
| legends[0].lineType   | String  | 路线样式类别（具体类型参考下图 type 枚举值）                                                           | 支持   | 支持   |
| legends[0].lineSpeed  | Number  | 路线流动速度（单位：米/秒）                                                                            | 支持   | 支持   |
| legends[0].curvature  | Number  | 路线曲率系数 （取值范围-1 ~ 1）                                                                        | 支持   | 支持   |
| legends[0].value      | Number  | 数据点权值                                                                                             | 支持   | 支持   |
| data                  | Array   | 定义图层可视化数据                                                                                     | 支持   | 支持   |
| legends[0].id         | String  | 图层子对象 id，新建时调用者自己传入的唯一标识，用于各种操作的对象识别                                  | 支持   | 支持   |
| legends[0].label      | String  | 图标标签文本                                                                                           | 支持   | 支持   |
| legends[0].time       | String  | 数据点时间戳                                                                                           | 支持   | 支持   |
| data[0].coord         | Array   | XY 轴坐标，X：经度；Y 纬度                                                                             | 支持   | 支持   |
| data[0].coordZ        | Number  | Z 轴高度（单位：米）                                                                                   | 支持   | 支持   |
| data[0].type          | String  | 数据点图例类别                                                                                         | 支持   | 支持   |

#### 回调数据

```json
{
  "result": 1,
  "message": "成功。"
}
```

#### 回调数据参数描述

| 属性    | 类型   | 说明                     |
| ------- | ------ | ------------------------ |
| result  | Number | 1：代表成功；0：代表失败 |
| message | String | 成功或者失败原因         |

## 3.更新点迹图样式

### 功能描述

更新三维场景中添加点迹图的样式

### 调用方法

```js
let jsonData = {
  id: 'trackLayerId',
  isAppend: false,
  legends: [
    {
      name: 'legendName1',
      icon: 'people',
      iconScale: 1,
      labelScale: 1,
      lineColor: '#FF00ff',
      lineWidth: 100,
      lineType: 'default',
      lineSpeed: 5,
      curvature: 1,
    },
    {
      name: 'legendName2',
      icon: 'bus',
      iconScale: 1,
      labelScale: 1,
      lineColor: '#00ff00',
      lineWidth: 100,
      lineType: 'arrow',
      lineSpeed: 20,
      curvature: 1,
    },
  ],
};
appInstance.uniCall('updateTrackLayerStyle', jsonData, (result) => {
  console.log(result);
});
```

#### 参数描述

| 属性                  | 类型    | 说明                                                                                                   |
| --------------------- | ------- | ------------------------------------------------------------------------------------------------------ |
| id                    | String  | 图层对象 id，新建时调用者自己传入的唯一标识，用于各种操作的对象识别                                    |
| isAppend              | Boolean | 是否追加数据（按数据顺序），true：在原有数据后方追加数据；false：清除原有数据，更新数据                |
| legends               | Array   | 定义图层包含图例，支持为不同图例定义各自样式                                                           |
| legends[0].name       | String  | 图层内图例名称，新建时调用者自己传入的唯一名称，用于各种操作的图例识别                                 |
| legends[0].icon       | String  | 内置图标名称，或根据项目，在场景中预置好的图标对象的名称，包含文字样式定义，具体图标名称参考上表和上图 |

| legends[0].lineWidth  | Number  | 路线宽度（单位：米）                                                                                   |
| legends[0].lineType   | String  | 路线样式类别（具体类型参考上图 type 枚举值）                                                           |

| legends[0].curvature  | Number  | 路线曲率系数（取值范围-1 ~ 1）                                                                         |
