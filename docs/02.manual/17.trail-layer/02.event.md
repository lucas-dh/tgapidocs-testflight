## 1.覆盖物单选事件

### 功能描述

开始覆盖物单选事件

### 调用方法

```js
appInstance.uniCall(
  'pickOverlay',
  {
    overlayType: 'trailLayer',
    idLayer: '0',
    type: 'click',
    isShowDecorator: true,
  },
  (result) => {
    console.log(result);
  }
);
```

#### 参数描述

| 属性            | 类型    | 说明                                                                |
| --------------- | ------- | ------------------------------------------------------------------- |
| overlayType     | String  | 可选中图层对象类别                                                  |
| idLayer         | String  | 填写图层对象 id，如不限制，可不填                                   |
| type            | String  | 单选模式，click：点击选中；hover：悬停选中                          |
| isShowDecorator | Boolean | 表示是否显示选中的高亮装饰符，true：显示，false：不显示，默认：true |

#### 回调数据

```json
{
  "result": 1,
  "message": "成功。"
}
```

#### 回调数据参数描述

| 属性    | 类型   | 说明                     |
| ------- | ------ | ------------------------ |
| result  | Number | 1：代表成功；0：代表失败 |
| message | String | 成功或者失败原因         |

## 2.开始轨迹图对象聚焦

### 功能描述

开始轨迹图对象聚焦

### 调用方法

```js
appInstance.uniCall(
  'addEventListener',
  {
    eventName: 'onFocusTrailLayerStart',
    callback: function (event) {
      console.log(event);
    },
  },
  (result) => {
    console.log(result);
  }
);
```

#### 回调数据

```json
{
  "idLayer": "layerId"
}
```

#### 回调数据参数描述

| 属性    | 类型   | 说明          |
| ------- | ------ | ------------- |
| idLayer | String | 轨迹图图层 id |

```js
appInstance.uniCall(
  'addEventListener',
  {
    eventName: 'onTrailLayerClick',
    callback: function (event) {
      console.log(event);
    },
  },
  (result) => {
    console.log(result);
  }
);
```

#### 回调数据

```json
{
  "idObj": "objId",
  "idLayer": "layerId",
  "type": "lengendName1",
  "label": "poi title",
  "selected": false
}
```

#### 回调数据参数描述

| 属性     | 类型    | 说明                     |
| -------- | ------- | ------------------------ |
| idLayer  | String  | 轨迹图层 id              |
| idObj    | String  | 轨迹图层对象 id          |
| type     | String  | 轨迹图层对象图例         |
| label    | String  | 轨迹图层对象标签         |
| selected | Boolean | 轨迹图层对象当前选中状态 |
