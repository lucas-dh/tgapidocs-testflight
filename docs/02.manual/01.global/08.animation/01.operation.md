### 功能描述

播放动画

### 调用方法

```js
let jsonData = {
  animationName: '蓝图动画名称',
  modelId: '模型ID',
  state: '开',
};

appInstance.uniCall('playAnimation', jsonData, (result) => {
  console.log(result);
});
```

#### 参数描述

| 属性          | 类型   | 说明                                                             |
| ------------- | ------ | ---------------------------------------------------------------- |
| animationName | String | 动画名称，与场景中的一致                                         |
| modelId       | String | 模型 ID                                                          |
| state         | String | 动画参数，会传给动画，与动画预制的状态匹配，表达例如开、关等状态 |

#### 回调数据

```js
{
  result: 1,
  message: '成功。',
}
```

#### 回调数据参数描述

| 属性    | 类型   | 说明                     |
| ------- | ------ | ------------------------ |
| result  | Number | 1：代表成功；0：代表失败 |
| message | String | 成功或者失败原因         |
