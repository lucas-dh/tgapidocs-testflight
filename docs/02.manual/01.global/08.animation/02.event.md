## 1.开始播放全局动画 <span style="width: 40px;height: 21px;display:inline-block;background: #0fcccc;margin-left: 5px;line-height: 19px;"><span style="width: 100%;height: 100%;background: url(./img/stream.png) no-repeat 4px 5px;background-size: 12px 10px;text-indent: 22px;font-size: 12px;color: #ffffff;line-height: 20px;display:inline-block;vertical-align: top;">流</span></span>


### 功能描述

开始播放全局动画

### 调用方法

```js
appInstance.uniCall(
  'addEventListener',
  {
    eventName: 'onPlayAnimationStart',
    callback: function (event) {
      console.log(event);
    },
  },
  (result) => {
    console.log(result);
  }
);
```

#### 回调数据

```json
{   
  "id": "animationName01",
  "modelId": "model01"
}

```

#### 回调数据参数描述

| 属性 | 类型   | 说明    |
| ---- | ------ | ------- |
| id   | String | 模型动画 id，新建时调用者自己传入的唯一标识，用于各种操作的对象识别 |
| modelId   | String | 如果播放时关联了模型Id，回调时也要传出 |
