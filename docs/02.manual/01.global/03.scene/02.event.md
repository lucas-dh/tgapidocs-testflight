### 功能描述

切换场景成功

### 调用方法

```js
appInstance.uniCall(
  'addEventListener',
  {
    eventName: 'onSceneSwitch',
    callback: function (event) {
      console.log(event);
    },
  },
  (result) => {
    console.log(result);
  }
);
```

#### 回调数据

```json
{   "name": "stateName",
    "default": false
}
```

#### 回调数据参数描述

| 属性    | 类型    | 说明               |
| ------- | ------- | ------------------ |
| name    | String  | 场景名称           |
| default | Boolean | 表示是不是默认场景 |
