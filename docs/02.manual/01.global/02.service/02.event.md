### 功能描述

链接服务成功

### 调用方法

```js
appInstance.uniCall(
  'addEventListener',
  {
    eventName: 'onServiceConnected',
    callback: function (event) {
      console.log(event);
    },
  },
  (result) => {
    console.log(result);
  }
);
```

## 5.服务断开

### 功能描述

服务断开

### 调用方法

```js
appInstance.uniCall(
  'addEventListener',
  {
    eventName: 'onServiceDisconnected',
    callback: function (event) {
      console.log(event);
    },
  },
  (result) => {
    console.log(result);
  }
);
```

#### 回调数据

```json
{
  "name": "serviceName",
  "code": "",
  "message": ""
}
```

#### 回调数据参数描述

| 属性    | 类型   | 说明                                                                                                                                            |
| ------- | ------ | ----------------------------------------------------------------------------------------------------------------------------------------------- |
| name    | String | 场景名称                                                                                                                                        |
| code    | String | 消息代码：消息内容。201: 服务器端被管理员终止。202: 使用时间已到 XXX 分钟，请稍后再试。203: 服务器进程异常，已退出。204: 和服务器网络连接异常。 |
| message | String | 消息内容                                                                                                                                        |
