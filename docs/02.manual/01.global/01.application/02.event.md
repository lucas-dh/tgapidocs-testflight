## 1.组件分辨率改变

### 功能描述

组件分辨率改变

### 调用方法

```js
appInstance.uniCall(
  'addEventListener',
  {
    eventName: 'onResolutionChange',
    callback: function (event) {
      console.log(event);
    },
  },
  (result) => {
    console.log(result);
  }
);
```

#### 回调数据

```json
{
  "width": 1920, // 新的 client 分辨率 宽
  "height": 1080 // 新的 client 分辨率 高
}
```

#### 回调数据参数描述

| 属性   | 类型   | 说明                  |
| ------ | ------ | --------------------- |
| width  | Number | 新的 client 分辨率 宽 |
| height | Number | 新的 client 分辨率 高 |

## 4.场景点击

### 功能描述

场景点击

### 调用方法

```js
appInstance.uniCall(
  'addEventListener',
  {
    eventName: 'onSceneClick',
    callback: function (event) {
      console.log(event);
    },
  },
  (result) => {
    console.log(result);
  }
);
```

#### 回调数据

```json
{
  "data": [
    {
      "coordType": 0,
      "coord": [113.95145617868572, 22.546972139271713],
      "coordZ": -0.1537687988281249
    },
    {
      "coordType": 1,
      "coord": [771.975521549718, -3478.5673736919653],
      "coordZ": 63.720703125
    }
  ],
  "source": "leftButtonUp",
  "clickObjType": "iconModel"
}
```

#### 回调数据参数描述

| 属性              | 类型   | 说明                                                                                                                  |
| ----------------- | ------ | --------------------------------------------------------------------------------------------------------------------- |
| data              | Array  | 触发事件时传递的参数对象                                                                                              |
| data[0].coordType | Number | 经纬度格式数据确认属性、属性值固定为 0                                                                                |
| data[0].coord     | Array  | 经度数据、纬度数据                                                                                                    |
| data[0].coordZ    | Number | 高度数据                                                                                                              |
| data[1].coordType | Number | 直角坐标格式数据确认属性、属性值固定为 1                                                                              |
| data[1].coord     | Array  | x 轴，y 轴 数据值                                                                                                     |
| data[1].coordZ    | Number | z 轴 数据值                                                                                                           |
| source            | String | 触发场景点击的设备名称，"leftButtonUp"：鼠标左键抬起触发或者触控设备上单指抬起触发，"rightButtonUp": 鼠标右键抬起触发 |
| clickObjType      | String | 点击的对象类型，可能是 "model", "iconModel", "building", "connection", "scene"                                        |
