## 1.设置环境天气

### 环境天气

#### 功能描述

设置环境天气

#### 调用方法

```js
let jsonData = {
  envWeather: 'Sunny',
};

appInstance.uniCall('setEnvWeather', jsonData, (result) => {
  console.log(result);
});
```

| envWeather | String | 天气类型（具体类型参考下图 envWeather 枚举值） | 只支持 sunny，snow，Rain | 支持   |

{

| 属性    | 类型   | 说明                     |
| ------- | ------ | ------------------------ |
| result  | Number | 1：代表成功；0：代表失败 |
| message | String | 成功或者失败原因         |

## 2.设置环境光照

### 环境光照

#### 功能描述

设置环境光照

#### 调用方法

```js
let jsonData = {
  envTime: '15:45',
  envDate: '2024/1/31',
  envTimeZone: 'UTC+8',
  fixedTime: false,
  alwayForward: false,
  duration: 5,
};

appInstance.uniCall('setEnvTime', jsonData, (result) => {
  console.log(result);
});
```

#### 参数描述

| 属性         | 类型    | 说明                                                                                                                                      | 端渲染 | 流渲染 |
| ------------ | ------- | ----------------------------------------------------------------------------------------------------------------------------------------- | ------ | ------ |
| envTime      | String  | 设置时间，结合 envDate 属性和 setBaseCenter 接口，可以显示正确的日照角度                                                                  | 支持   | 支持   |
| envDate      | String  | 设置日期，结合 envTime 属性和 setBaseCenter 接口，可以显示正确的日照角度                                                                  | 不支持 | 支持   |
| envTimeZone  | String  | 设置时区，默认值：UTC+8，可取值：UTC、UTC+1 至 UTC+12、UTC-1 至 UTC-12                                                                    | 不支持 | 支持   |
| fixedTime    | Boolean | 时间是否固定，false：默认值，会从用户设置的时间，一秒一秒往后走，若走到晚上，则场景会变成黑夜效果，true：场景中的时间一直是用户设置的时间 | 不支持 | 支持   |
| alwayForward | String  | 时间是否总是向前，false：如果新时间小于当前时间，时间会倒回去，true：如果新时间小于当前时间，会经历 23:59 分再到新时间                    | 支持   | 支持   |
| duration     | Number  | 切换时间，秒                                                                                                                              | 支持   | 支持   |

#### 回调数据

```json
{
  "result": 1,
  "message": "成功。"
}
```

#### 回调数据参数描述

| 属性    | 类型   | 说明                     |
| ------- | ------ | ------------------------ |
| result  | Number | 1：代表成功；0：代表失败 |
| message | String | 成功或者失败原因         |

| ------- | ------ | ------------------------ |

## 13.设置对象交互轴

### 功能描述

设置对象交互轴

#### 调用方法

```js
let jsonData = {
  overlayType: 'landmark',
  idLayer: 'landmarkId',
  id: 'landmarkId',
  ids: ['Car03', 'Car01'],
  relative: 'self',
  axisType: 'transform',
  directionType: ['all'],
  unit: 1,
};
appInstance.uniCall('showObjectAxis', jsonData, (result) => {
  console.log(result);
});
```

#### 参数描述

| 属性          | 类型   | 说明                                                                                                                                                                                                                                    | 端渲染                        | 流渲染 |
| ------------- | ------ | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ----------------------------- | ------ |
| overlayType   | String | 目前支持的 overlayType 是：landmark，path，area，circularArea，model，3DMarker，modelTrailLayer，modelLandmarkLayer，iconModelTrailLayer，iconModelLandmarkLayer，landmarkLayer                                                         | 不支持 3DMarker landmarkLayer | 支持   |
| idLayer       | String | 如果是 landmark，path，area，circularArea，3DMarker（只支持流），modelTrailLayer，modelLandmarkLayer，iconModelTrailLayer，iconModelLandmarkLayer，landmarkLayer 类型的 overlayType，需要图层 Id，如果是 model 的 overlayType，可以为空 | 不支持 3DMarker landmarkLayer | 支持   |
| id            | String | 如果是 landmark，path，area，modelTrailLayer，modelLandmarkLayer，iconModelTrailLayer，iconModelLandmarkLayer，landmarkLayer 类型的 overlayType，是图层子对象 id，如果是 model 类型的 overlayType，就是模型 id                          | 不支持 landmarkLayer          | 支持   |
| ids           | Array  | 如果希望有多个相同类型的对象显示交互轴，通过数组传入全部 ID，这些 ID 会和单独 id 属性合并成处理                                                                                                                                         | 支持                          | 支持   |
