## 1.平移交互轴拖拽事件

### 功能描述

平移交互轴拖拽事件

### 调用方法

```js
appInstance.uniCall(
  'addEventListener',
  {
    eventName: 'onTransformAxisDrag',
    callback: function (event) {
      console.log(event);
    },
  },
  (result) => {
    console.log(result);
  }
);
```

#### 回调数据

```json
{
  "idObj": "landmarkId",
  "idLayer": "landmarkId",
  "overlayType": "landmark",
  "coordType": 0,
  "data": [
    {
      "coord": [113.93424336592119, 22.517487787024894],
      "coordZ": 63.99999999999998
    }
  ]
}
```

#### 回调数据参数描述

| 属性           | 类型   | 说明                                                                                                                                                                       | 端渲染          | 流渲染 |
| -------------- | ------ | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | --------------- | ------ |
| idObj          | String | 图层对象 id，新建时调用者自己传入的唯一标识，用于各种操作的对象识别                                                                                                        | 支持            | 支持   |
| idLayer        | String | 图层对象 id，新建时调用者自己传入的唯一标识，用于各种操作的对象识别                                                                                                        | 支持            | 支持   |
| overlayType    | String | 图层类别 目前支持的 overlayType 是：landmark，path，area， circularArea，model，3DMarker，modelTrailLayer，modelLandmarkLayer，iconModelTrailLayer，iconModelLandmarkLayer | 不支持 3DMarker | 支持   |
| coordType      | Number | XY 轴坐标类别，0：X 经度，Y 纬度；1：X 米，Y 米，相对世界 00 点                                                                                                            | 支持            | 支持   |
| data           | Array  | 图层数据                                                                                                                                                                   | 支持            | 支持   |
| data[0].coord  | Array  | XY 轴坐标，X：经度；Y：纬度                                                                                                                                                | 支持            | 支持   |
| data[0].coordZ | Number | Z 轴高度（单位：米）                                                                                                                                                       | 支持            | 支持   |

## 2.缩放交互轴拖拽事件

缩放交互轴拖拽事件

```js
appInstance.uniCall(
  'addEventListener',

      console.log(event);
    },
  },
  (result) => {
    console.log(result);
  }
);
```

```json
{
  "idObj": "objId",
  "idLayer": "layherId",
  "overlayType": "landmark",
  "coordType": 0,

      "scale": [1, 1, 1],
      "name": "modelName"
    },
    {
      "scale": [1, 1, 1],

  ]
}
```

## 3.旋转交互轴拖拽事件

旋转交互轴拖拽事件

### 调用方法

```js
appInstance.uniCall(
  'addEventListener',
  {
    eventName: 'onRotationAxisDrag',
    callback: function (event) {
      console.log(event);
    },
  },
  (result) => {
    console.log(result);
  }
);
```

#### 回调数据

```json
{
  "idObj": "objId",
  "idLayer": "layherId",
  "overlayType": "landmark",
  "coordType": 0,
  "data": [
    {
      "rotation": [0, 0, 0],
      "name": "modelName"
    },
    {
      "rotation": [0, 0, 0],
      "name": "modelName"
    }
  ]
}
```

| overlayType      | String | 图层类别 目前支持的 overlayType 是：landmark，path，area， circularArea，model，3DMarker，modelTrailLayer，modelLandmarkLayer，iconModelTrailLayer，iconModelLandmarkLayer                                                                                                                                                                             | 不支持 3DMarker | 支持   |
