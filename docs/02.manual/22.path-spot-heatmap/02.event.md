## 1.覆盖物单选事件

### 功能描述

开始覆盖物单选事件

### 调用方法

```js
appInstance.uniCall(
  'pickOverlay',
  {
    overlayType: 'roadPtHeatLayer',
    idLayer: '0',
    type: 'click',
    isShowDecorator: true,
  },
  (result) => {
    console.log(result);
  }
);
```

#### 参数描述

| 属性            | 类型    | 说明                                                                |
| --------------- | ------- | ------------------------------------------------------------------- |
| overlayType     | String  | 可选中图层对象类别                                                  |
| idLayer         | String  | 填写图层对象 id，如不限制，可不填                                   |
| type            | String  | 单选模式，click：点击选中；hover：悬停选中                          |
| isShowDecorator | Boolean | 表示是否显示选中的高亮装饰符，true：显示，false：不显示，默认：true |

#### 回调数据

```json
{
  "result": 1,
  "message": "成功。"
}
```

#### 回调数据参数描述

| 属性    | 类型   | 说明                     |
| ------- | ------ | ------------------------ |
| result  | Number | 1：代表成功；0：代表失败 |
| message | String | 成功或者失败原因         |

## 2.开始路径点热力图对象聚焦

### 功能描述

开始路径点热力图对象聚焦

### 调用方法

```js
appInstance.uniCall(
  'addEventListener',
  {
    eventName: 'onFocusRoadPtHeatLayerStart',
    callback: function (event) {
      console.log(event);
    },
  },
  (result) => {
    console.log(result);
  }
);
```

#### 回调数据

```json
{
  "idLayer": "layerId"
}
```

#### 回调数据参数描述

| 属性    | 类型   | 说明              |
| ------- | ------ | ----------------- |
| idLayer | String | 路径点热力图层 id |

## 6.鼠标悬停路径点热力图对象

### 功能描述

鼠标悬停路径点热力图对象

### 调用方法

```js
appInstance.uniCall(
  'addEventListener',
  {
    eventName: 'onRoadPtHeatLayerHover',
    callback: function (event) {
      console.log(event);
    },
  },
  (result) => {
    console.log(result);
  }
);
```

#### 回调数据

```json
{
  "idLayer": "layerId",
  "coord": [113.320381, 23.108027]
}
```

#### 回调数据参数描述

| 属性  | 类型   | 说明                                 |
| ----- | ------ | ------------------------------------ |
| idObj | String | 鼠标进入路径点对象时触发此事件       |
| coord | Array  | 鼠标进入路径点对象时的第一个交点位置 |

## 8.左键点击路径点热力图对象

### 功能描述

左键点击路径点热力图对象

### 调用方法

```js
appInstance.uniCall(
  'addEventListener',
  {
    eventName: 'onRoadPtHeatLayerClick',
    callback: function (event) {
      console.log(event);
    },
  },
  (result) => {
    console.log(result);
  }
);
```

#### 回调数据

```json
{
  "idLayer": "layerId",
  "selected": false, // 选中 true，取消选中 false
  "coord": [113.320381, 23.108027] // 鼠标点击路径点对象时的鼠标位置
}
```

#### 回调数据参数描述

| 属性    | 类型    | 说明                           |
| ------- | ------- | ------------------------------ |
| idObj   | String  | 鼠标离开路径点对象时触发此事件 |
| idLayer | Boolean | 路径点热力图层 id              |
| coord   | Array   | 路径点热力图层对象当前选中状态 |
