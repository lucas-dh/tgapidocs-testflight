## 1.覆盖物单选事件

### 功能描述

开始覆盖物单选事件

### 调用方法

```js
appInstance.uniCall(
  'pickOverlay',
  {
    overlayType: 'path',
    idLayer: '0',
    type: 'click',
    isShowDecorator: true,
  },
  (result) => {
    console.log(result);
  }
);
```

#### 参数描述

| 属性            | 类型    | 说明                                                                |
| --------------- | ------- | ------------------------------------------------------------------- |
| overlayType     | String  | 可选中图层对象类别                                                  |
| idLayer         | String  | 填写图层对象 id，如不限制，可不填                                   |
| type            | String  | 单选模式，click：点击选中；hover：悬停选中                          |
| isShowDecorator | Boolean | 表示是否显示选中的高亮装饰符，true：显示，false：不显示，默认：true |

#### 回调数据

```json
{
  "result": 1,
  "message": "成功。"
}
```

#### 回调数据参数描述

| 属性    | 类型   | 说明                     |
| ------- | ------ | ------------------------ |
| result  | Number | 1：代表成功；0：代表失败 |
| message | String | 成功或者失败原因         |

## 2.开始路径对象聚焦

### 功能描述

开始路径对象聚焦

### 调用方法

```js
appInstance.uniCall(
  'addEventListener',
  {
    eventName: 'onFocusPathStart',
    callback: function (event) {
      console.log(event);
    },
  },
  (result) => {
    console.log(result);
  }
);
```

#### 回调数据

```json
{
  "id": "objId"
}
```

#### 回调数据参数描述

| 属性 | 类型   | 说明        |
| ---- | ------ | ----------- |
| id   | String | 路径图层 id |

鼠标悬停路径对象

### 调用方法

```js
appInstance.uniCall(
  'addEventListener',
  {
    eventName: 'onPathHover',
    callback: function (event) {
      console.log(event);
    },
  },
  (result) => {
    console.log(result);
  }
);
```

#### 回调数据

```json
{
  "id": "objId",
  "tag": "customerinfo",
  "coord": [113.320381, 23.108027]
}
```

#### 回调数据参数描述

| 属性  | 类型     | 说明                               |
| ----- | -------- | ---------------------------------- |
| id    | String   | 路径图层 id                        |
| tag   | Anything | 用户自定义属性                     |
| coord | Array    | 鼠标进入路径对象时的第一个交点位置 |

| 属性     | 类型     | 说明                         |
| -------- | -------- | ---------------------------- |
