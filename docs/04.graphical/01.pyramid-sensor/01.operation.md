## 1.添加四棱锥体 <span style="width: 40px;height: 21px;display:inline-block;background: #0fcccc;margin-left: 5px;line-height: 19px;"><span style="width: 100%;height: 100%;background: url(./img/stream.png) no-repeat 4px 5px;background-size: 12px 10px;text-indent: 22px;font-size: 12px;color: #ffffff;line-height: 20px;display:inline-block;vertical-align: top;">流</span></span>

### 功能描述

在三维场景中添加四棱锥体

### 调用方法

```js
let jsonData = {
  id: 'pyramidSensor',
  coordType: 0,	
  coordTypeZ: 0,
  coord: [116.34599081794,40.08430867971],
  coordZ: 30,
  visible: true,
  angleH: 10,
  angleV: 10,
  length: 10,
  rotation: [20,80,90],
  alpha: 0.5,
  color: '#ffffff',
  outline: true,
  outlineColor: '#ffffff',
  topSurface: true,
  topOutline: true,
  topStepsV: 5,
  topStepsH: 8,	
  topStyle: 'flat',	
  showScanner: true,
  scannerColor: "#00ffff",
  scannerAlpha: 0.9,
  scannerMode: 'vertical',
  scannerDuration: 3,
  scannerRepeatMode: 'backward',
  lookAtObj: '03_FACS_Billboard_L_179',
};

appInstance.uniCall('addPyramidSensor', jsonData, (result) => {
  console.log(result);
});
```

#### 参数描述

| 属性             | 类型     | 说明                                                                                                                                      
| ---------------- | -------- | ----------------------------------------------------------------------------------------------------------------------------------------- 
| id               | String   | 图层对象 id，新建时调用者自己传入的唯一标识，用于各种操作的对象识别                                                                         
| coordType        | Number   | XY 轴坐标类别，0：X 经度，Y 纬度；1：X 米，Y米，相对世界00点                                                                         
| coordTypeZ       | Number   | Z 轴坐标类别，0：相对 3D 世界 0 海拔；1：相对 3D 世界地表；2：相对 3D 模型表面（单位：米）                                             
| coord            | Array    | XY 轴坐标，X：经度；Y：纬度                                                                                                          
| coordZ           | Number   | Z 轴高度（单位：米）                                                                                                                 
| visible          | Boolean  | 初始化添加时的默认值，true：显示，false：隐藏                                                                                       
| angleH           | Number   | 四棱锥横向夹角，（单位：角度，0.1 - 180）                                                                                         
| angleV           | Number   | 四棱锥纵向夹角，（单位：角度，0.1 - 360）                                                                                     
| length           | Number   | 四棱锥半径长度，（单位：米），当lookAtObj凝视目标不为空时，这个表示从凝视目标继续延长出的长度 
| rotation         | Array    | XYZ 三轴旋转度数 [偏航角（单位：角度，0正北, -360 ~ 360）, 俯仰角（单位：角度，0水平, -360 ~ 360）,翻滚角（单位：角度，0水平, -360 ~ 360）]       
| alpha            | Number   | 透明度，0：完全透明；1：完全不透明                                                                                         
| color            | String   | 四棱锥颜色                                                                                        
| outline          | Boolean  | 是否显示描边                                                                                              
| outlineColor     | String   | 描边颜色                                                                                                       
| topSurface       | Boolean  | 是否显示顶面                                                                                                     
| topOutline       | Boolean  | 是否显示顶面网格线                                                                                                     
| topStepsV        | Number   | 顶面网格线纵向数量                                                                                                     
| topStepsH        | Number   | 顶面网格线横向数量                                                                                                     
| topStyle         | String   | 顶面的样式，flat：平面，curved：曲面                                                                                                     
| showScanner      | Boolean  | 是否显示扫描面                                                                                                 
| scannerColor     | String   | 扫描面颜色                                                                                                 
| scannerAlpha     | Number   | 扫描面透明度，0：完全透明；1：完全不透明                                                                                                 
| scannerMode      | String   | 扫描模式，vertical：垂直方向，horizontal：水平方向，subCycle：子锥体循环，subRandom：子锥体随机                                                                                                 
| scannerDuration  | Number   | 扫描周期，一个周期所需的时间，（单位：秒）                                                                                                
| scannerRepeatMode  | String   | 扫描重复模式，backward：反向开始，startover：从头开始                                                                                               
| lookAtObj  | String   | 凝视目标的 对象Id。当前锥体rotation和半径会跟随目标坐标变化，自身位置不变。对象可以是model、building或者landmarker                                                                                                        

#### 回调数据

```json
{
  "result": 1,
  "message": "成功。"
}
```

#### 回调数据参数描述

| 属性    | 类型   | 说明                     |
| ------- | ------ | ------------------------ |
| result  | Number | 1：代表成功；0：代表失败 |
| message | String | 成功或者失败原因         |
