## 1.设置天际线 <span style="width: 40px;height: 21px;display:inline-block;background: #0fcccc;margin-left: 5px;line-height: 19px;"><span style="width: 100%;height: 100%;background: url(./img/stream.png) no-repeat 4px 5px;background-size: 12px 10px;text-indent: 22px;font-size: 12px;color: #ffffff;line-height: 20px;display:inline-block;vertical-align: top;">流</span></span>

### 功能描述

在三维场景中设置天际线

### 调用方法

```js
let jsonData = {
  visible: true,
  width: 2,
  color: '#ff0000',
};

appInstance.uniCall('setSkyline', jsonData, (result) => {
  console.log(result);
});
```

#### 参数描述

| 属性             | 类型     | 说明                                                                                                                                      
| ---------------- | -------- | ----------------------------------------------------------------------------------------------------------------------------------------- 
| visible          | Boolean  | 设置天际线的显示状态，true：显示，false：隐藏                                                                                                
| width            | Number   | 天际线宽度（单位：像素）                                                                                                                    
| color            | String   | 天际线的颜色                                                                                                                               

#### 回调数据

```json
{
  "result": 1,
  "message": "成功。"
}
```

#### 回调数据参数描述

| 属性    | 类型   | 说明                     |
| ------- | ------ | ------------------------ |
| result  | Number | 1：代表成功；0：代表失败 |
| message | String | 成功或者失败原因         |
