## 1.设置分屏比对 <span style="width: 40px;height: 21px;display:inline-block;background: #0fcccc;margin-left: 5px;line-height: 19px;"><span style="width: 100%;height: 100%;background: url(./img/stream.png) no-repeat 4px 5px;background-size: 12px 10px;text-indent: 22px;font-size: 12px;color: #ffffff;line-height: 20px;display:inline-block;vertical-align: top;">流</span></span>

### 功能描述

将屏幕划分成两个区域，比对添加图层前后或者控制图层显隐后的差别

### 调用方法

```js
let jsonData = {
  visible: true,
  orientation: 'v',
};

appInstance.uniCall('setSplitCompare', jsonData, (result) => {
  console.log(result);
});
```

#### 参数描述

| 属性        | 类型    | 说明                                                                                                    |
| ----------- | ------- | ------------------------------------------------------------------------------------------------------- |
| visible     | Boolean | 设置分屏比对显示状态，true：开启，false：关闭，关闭时窗口使用左侧屏幕状态                               |
| orientation | String  | 分屏方向，"v"：垂直分为左右两个屏幕，"h"：水平分为上下两个屏幕，"vroller"：垂直分为左右两个屏幕卷帘比对 |

#### 回调数据

```json
{
  "result": 1,
  "message": "成功。"
}
```

#### 回调数据参数描述

| 属性    | 类型   | 说明                     |
| ------- | ------ | ------------------------ |
| result  | Number | 1：代表成功；0：代表失败 |
| message | String | 成功或者失败原因         |
