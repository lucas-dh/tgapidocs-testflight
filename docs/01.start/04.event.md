## 1.添加事件监听

### 功能描述

添加事件监听

### 调用方法

```js
appInstance.uniCall(
  'addEventListener',
  {
    eventName: 'onLandmarkHover',
    callback: function (event) {
      console.log(event);
    },
  },
  (result) => {
    console.log(result);
  }
);
```

#### 参数描述

| 属性      | 类型     | 说明                                                                                                                                                                         |
| --------- | -------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| eventName | String   | 事件名称                                                                                                                                                                     |
| callback  | Function | 要注册的事件监听回调函数，可对同一个事件注册多个回调函数。注意这个回调函数对应的是第一个参数表示的事件发生时的回调函数，要和本次 addEventListener 调用完成的回调函数区分开。 |

## 2.移除事件监听

### 功能描述

移除事件监听

### 调用方法

```js
appInstance.uniCall(
  'removeEventListener',
  {
    eventName: 'onLandmarkHover',
    callback: window.eventCallback,
  },
  window.eventCallback
);
```

#### 参数描述

| 属性      | 类型     | 说明                                                                                                                                                                         |
| --------- | -------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| eventName | String   | 事件名称                                                                                                                                                                     |
| callback  | Function | 要注销的事件监听回调函数，此处的回调函数不能是匿名函数。如果不提供本参数，则注销指定事件上的所有回调函数。注意这个回调函数要和本次 addEventListener 调用完成的回调函数区分开。 |
