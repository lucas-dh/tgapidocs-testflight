## 1.开始视频融合对象聚焦

### 功能描述

开始视频融合对象聚焦

### 调用方法

```js
appInstance.uniCall(
  'addEventListener',
  {
    eventName: 'onFocusVideoFusionStart',
    callback: function (event) {
      console.log(event);
    },
  },
  (result) => {
    console.log(result);
  }
);
```

#### 回调数据

```json
{
  "id": "objId1"
}
```

#### 回调数据参数描述

| 属性 | 类型   | 说明                      |
| ---- | ------ | ------------------------- |
| id   | String | 开始聚焦的视频融合对象 id |
