## 1.鼠标悬停建筑对象

### 功能描述

鼠标悬停建筑对象

### 调用方法

```js
appInstance.uniCall(
  'addEventListener',
  {
    eventName: 'onBuildingHover',
    callback: function (event) {
      console.log(event);
    },
  },
  (result) => {
    console.log(result);
  }
);
```

#### 回调数据

```json
{
  "id": "objId"
}
```

#### 回调数据参数描述

| 属性 | 类型   | 说明        |
| ---- | ------ | ----------- |
| id   | String | 建筑对象 ID |
