## 1.开始特效对象聚焦

### 功能描述

开始特效对象聚焦

### 调用方法

```js
appInstance.uniCall(
  'addEventListener',
  {
    eventName: 'onFocus3DMarkerStart',
    callback: function (event) {
      console.log(event);
    },
  },
  (result) => {
    console.log(result);
  }
);
```

#### 回调数据

```json
{
  "id": "objId"
}
```

#### 回调数据参数描述

| 属性 | 类型   | 说明        |
| ---- | ------ | ----------- |
| id   | String | 特效对象 id |
