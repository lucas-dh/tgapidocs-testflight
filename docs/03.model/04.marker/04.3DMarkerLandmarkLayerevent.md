## 1.开始特效地标图对象聚焦

### 功能描述

开始特效地标图对象聚焦

### 调用方法

```js
appInstance.uniCall(
  'addEventListener',
  {
    eventName: 'onFocus3DMarkerLandmarkLayerStart',
    callback: function (event) {
      console.log(event);
    },
  },
  (result) => {
    console.log(result);
  }
);
```

#### 回调数据

```json
{
  "idLayer": "layerId"
}
```

#### 回调数据参数描述

| 属性 | 类型   | 说明        |
| ---- | ------ | ----------- |
| idLayer   | String | 特效对象图层 id |
