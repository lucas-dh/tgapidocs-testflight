## 1.覆盖物单选事件

### 功能描述

开始覆盖物单选事件

### 调用方法

```js
appInstance.uniCall(
  'pickOverlay',
  {
    overlayType: 'iconmodelLandmarkLayer',
    idLayer: '0',
    type: 'click',
    isShowDecorator: true,
  },
  (result) => {
    console.log(result);
  }
);
```

#### 参数描述

| 属性            | 类型    | 说明                                                                |
| --------------- | ------- | ------------------------------------------------------------------- |
| overlayType     | String  | 图层类别 目前支持的overlayType是：landmark，path，area，model，modelTrailLayer，modelLandmarkLayer，iconModelTrailLayer，iconModelLandmarkLayer                                                  |
| idLayer         | String  | 填写图层对象 id，如不限制，可不填                                   |
| type            | String  | 单选模式，click：点击选中；hover：悬停选中                          |
| isShowDecorator | Boolean | 表示是否显示选中的高亮装饰符，true：显示，false：不显示，默认：true |

#### 回调数据

```json
{
  "result": 1,
  "message": "成功。"
}
```

#### 回调数据参数描述

| 属性    | 类型   | 说明                     |
| ------- | ------ | ------------------------ |
| result  | Number | 1：代表成功；0：代表失败 |
| message | String | 成功或者失败原因         |
